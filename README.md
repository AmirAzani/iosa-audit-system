# IOSA Audit Tracking System ✈️

A web-based system developed to manage IOSA (IATA Operational Safety Audit) records, audits, and findings efficiently.

## 🔧 Features
- Manage airline operators
- Track IOSA audit history
- Add and manage audit findings & observations
- Role-based access (Admin / User)
- Clean dashboard for audit overview

## 🛠️ Tech Stack
- Backend: Python (Flask)
- Database: MySQL
- Frontend: HTML, CSS (Jinja2 Templates)

## 📸 System Modules
- Operator Management
- Audit Year Tracking
- Findings & Observations
- Audit Status Monitoring

```bash
git clone https://github.com/AmirAzani/iosa-audit-system.git
